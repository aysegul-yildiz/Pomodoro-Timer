# Pomodoro-Timer
This is a pomodoro timer app that uses the tkinter module to create the graphical user interface. The app has a simple but nice design. The tomato image at the center has the timer displayed on it. There is start and reset buttons to start pomodoro. When user hits start button the work time begins, after 25 minute work time a check mark is displayed at the scrren and 5 minute break begins after short break another work session begins and after that new check mark is added to show how many work sessions have been completed. After 4 work sessions and short breaks a long break of 20 minutes starts. The reset button resets everything and allows user to start over.
